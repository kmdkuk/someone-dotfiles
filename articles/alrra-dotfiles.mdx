# alrra/dotfiles

Bash 使い。  
bashrc.local など個人設定でカスタマイズを読み込むような設計になっている。  
src/os/setup.sh が bootstrap script。Bash 製  
home 下に clone じゃなくて~/projects/dotfiles においておくことを想定している。  
Mac と Ubuntu 用。  
Mac は Homebrew, Ubuntu は apt-get  
かなり setup 用の script を分割して管理している。
