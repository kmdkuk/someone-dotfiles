# webpro/dotfiles

Mac がメイン、Ubuntu や Arch でも使えるようにしてしているよう。  
タスクランナーに Makefile を使っている。  
Bash 使い。  
Bash のテスティングフレームワーク bats を利用したテストが含まれている。  
Mac では Homebrew, Arch に pacman。VS Code の extensions もファイル管理されている。
