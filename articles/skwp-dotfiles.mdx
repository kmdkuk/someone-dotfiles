# skwp/dotfiles

install.sh が bootstrap script。対話的にセットアップするのにも対応している。  
タスクランナーに rake を使っている。  
普段の開発環境に Docker や Docker Compose を利用しているっぽい。  
Zsh で prezto 使い。  
Homebrew 利用だが、Mac のみで分岐している。おそらく Mac メイン使い。  
package は Rakefile の中に直接定義されている。  
Ruby/Rails 使いの方。
